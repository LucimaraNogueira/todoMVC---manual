📌 Sobre o Projeto

🔹Este repositório apresenta os testes manuais realizados na aplicação TodoMVC, uma aplicação web de gerenciamento de tarefas  **[TodoMVC (React Version)](https://todomvc.com/examples/react/dist/)**.


🎯 Objetivo

🔹Verificar o correto funcionamento das funcionalidades principais

🔹Análise de Requisitos: Identificação de fluxos críticos e regras de negócio.

🔹BDD (Behavior Driven Development):Escrita de cenários em Gherkin.

🔹Test Management Uso do MIRO como ferramenta de gestão de ciclo de vida dos testes (substituindo planilhas tradicionais).

🔹Identificar falhas, erros ou comportamentos inesperados

🔹Garantir uma boa experiência do usuário

🔹Praticar conceitos de QA (Quality Assurance) e testes manuais

## Documentação do Projeto (Live Dashboard)

Toda a estratégia, planejamento e execução foram documentados em um Dashboard Interativo no MIRO.
Isso permite uma visualização em tempo real do progresso, métricas e gestão de defeitos.

### [ACESSE A ANÁLISE DOS REQUISITOS NO MIRO AQUI](https://miro.com/app/board/uXjVIDJs63s=/)

> **O que você encontrará no Dashboard:**
> * 🗺️ **Estratégia:** Definição de escopo (In/Out), ambiente e critérios de aceite.
> * 🗃️ **Repositório de Testes:** Casos de teste detalhados com BDD.
> * 🏃‍♂️ **Kanban de Execução:** Visualização ágil do status dos testes.
> * 🐛 **Gestão de Bugs:** Relatórios de defeitos com evidências e severidade.

🔍 Escopo dos Testes

🔹Os testes foram executados nos principais fluxos do usuário:

🔹Criar novas tarefas

🔹Editar tarefas existentes

🔹Marcar tarefas como concluídas

🔹Excluir tarefas

🔹Utilizar filtros (Todas, Ativas e Concluídas)

🔹Verificar a persistência das tarefas após atualizar a página

🧪 Tipos de Testes Realizados

🔹Testes Funcionais

🔹Testes de Interface (UI)

🔹Testes de Usabilidade

🔹Testes Positivos e Negativos

🔹Testes de Validação de Campos

📝 Casos de Teste

🔹Foram criados casos de teste manuais contendo:

🔹ID do caso de teste

🔹Descrição do teste

🔹Pré-condições

🔹Passos para execução

🔹Resultado esperado

🔹Resultado obtido

🔹Status (Aprovado ou Reprovado)

🛠️ Ferramentas Utilizadas

🔹Aplicação TodoMVC

🔹Navegador Web (Opera)

🔹MIRO como ferramenta de gestão de ciclo de vida dos testes

🔹Planilhas ou documentos para organização dos testes
